## The Stack

- **Hardware**: 5 Raspberry Pi 5 + 2 Hailo-8 NPUs (52 TOPS) + 2 DigitalOcean droplets
- **Edge**: 496 Cloudflare Workers across 20 domains
- **Memory**: PS-SHA∞ append-only hash chain (adaptive depth 3-7)
- **Identity**: RoadID — sovereign, portable, persistent, exportable as JSON
- **Logic**: Trinary (+1/0/-1) with Z-minimization equilibrium
- **Blockchain**: RoadChain (D1 persistent, hash-linked blocks)
- **Token**: RoadCoin (1B supply, earn/spend/stake/transfer)
- **Payments**: Stripe (fiat) + Coinbase Commerce (crypto) + x402 (micropayments)
- **Mesh**: WireGuard + Tailscale
- **Self-hosted**: Gitea, MinIO, PostgreSQL, Redis, Ollama, PowerDNS, Caddy
- **Cost**: $136/month total infrastructure
